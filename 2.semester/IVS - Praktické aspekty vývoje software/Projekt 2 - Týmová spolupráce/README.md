# IVS Team Project

Požadavky
---------

- Python3
- pip3
- virtualenv
- TkInter
- Doxygen

Instalace
---------

Vytvoření virtuálního prostředí
```
$ virtualenv venv
```

Aktivace virtuálního prostředí
```
$ source venv/bin/activate
```

Instalace PyPI balíčků
```
$ pip install -r requirements.txt
```

Prostredi
---------

Windows 64bit
